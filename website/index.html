About Me

Hello! I'm Logan Godsey, a Senior Support Engineer.

From a young age I've always been interested in technology, whether it was about space, consumer, or enterprise level technology.

I've always wanted a fairly static website that I can host myself to show off some of the skills I've learned and this would be that place!

This website is hosted in a personal Kubernetes cluster located within my own network.

It is deployed using a custom Helm chart which is then built and deployed via CircleCI.

You can see the source code in my repository here.

Nginx Proxy Manager sits in front of my Kong deployment to provide SSL certs while Kong provides the routing to the web-ui pod.
